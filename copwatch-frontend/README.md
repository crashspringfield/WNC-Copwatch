# WNC Copwatch
Frontend application written in React/TypeScript. Interacts w/ Node via GraphQL. Leaflet and D3 to come.

## Getting Started
* `yarn install`
* `echo "SERVER_URL=http://localhost:4000" > .env` (or whatever)
* `npm start`
